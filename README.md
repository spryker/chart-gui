# ChartGui Module
[![Build Status](https://travis-ci.org/spryker/chart-gui.svg)](https://travis-ci.org/spryker/chart-gui)

ChartGui provides user interface to interact with charts in zed.

## Installation

```
composer require spryker/chart-gui
```
